### Exercise 1.09

Indicate which, if any, of the following output statements are legal:

    std::cout << "/*";
    std::cout << "*/";
    std::cout << /* "*/" */;
    std::cout << /* "*/" /* "/*" */;

### Solution

    std::cout << "/*";                  Legal
    std::cout << "*/";                  Legal
    std::cout << /* "*/" */";           Illegal - Fixed
    std::cout << /* "*/" /* "/*" */;    Legal
